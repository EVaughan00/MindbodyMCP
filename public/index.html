Mindbody MCP

Connect your Mindbody studio to Claude or ChatGPT. Follow the steps below to get started.

1

Get Mindbody API access

Pick whichever path fits you. Either way, the goal is a live Site ID connected to a developer account.

Path A · Self-register

Register your own developer account and activate your site on the Mindbody Developer Portal. When it's active, send me your API key and Site ID so I can register your studio here.

Path B · I activate you

Don't want to manage a developer account? Email me your Site ID and I'll activate your studio under my developer account using my existing API key — no portal signup needed.

Note: if you register through me, I'll invoice you for Mindbody API usage.

2

I register your studio

Once I have your details (Path A: your API key + Site ID · Path B: just your Site ID), I'll add your studio and send you a Tenant ID and password. These are your login for the connector below.

Questions or ready to start? Email evanvaughan@unboxedtechnology.com.

3

Connect Claude or ChatGPT

Add this MCP endpoint as a custom connector and sign in with the Tenant ID + password I gave you:

/mcp

Claude Pro, Max, Team, or Enterprise

  1. Open Claude (web or desktop app) and go to Settings → Connectors.
  2. Click Add custom connector and paste the MCP URL above.
  3. Connect and sign in with your Tenant ID + password.

ChatGPT Plus, Pro, Business, or Enterprise

  1. Go to Settings → Connectors (enable Developer mode if prompted).
  2. Click Add / Create, choose MCP, and paste the MCP URL above.
  3. Connect and sign in with your Tenant ID + password.
